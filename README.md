# Reciper App

Мобильный AI-ассистент питания, генерации рецептов и пошагового приготовления блюд. Цель приложения — избавить пользователя от рутины подсчета КБЖУ, выбора блюд и инвентаризации холодильника.

## 🚀 Возможности

- **AI Сканер**: Умное сканирование содержимого холодильника по фотографии.
- **Генерация рецептов**: Подбор вариантов блюд на основе найденных продуктов с расчетом КБЖУ.
- **Режим готовки**: Удобный пошаговый режим приготовления с таймерами.
- **Трекинг прогресса**: Отрисовка графиков и статистика потребления калорий и макронутриентов.

## 🛠️ Стек технологий

- **Frontend (Мобильное приложение)**: Flutter (Dart), Riverpod, Clean Architecture
- **Backend**: Python 3.11+, FastAPI, Clean Architecture
- **Database / Cache**: PostgreSQL, Redis
- **AI / Инфраструктура**: Локальные VLM модели (подобные LLaVA 1.5, Moondream2 через Ollama)
