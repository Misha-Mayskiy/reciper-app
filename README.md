# Reciper App

A comprehensive mobile application designed to act as a personal nutrition and cooking assistant. Reciper helps users make healthier food choices, discover new recipes, and improve their cooking skills through AI-powered guidance.

## 🚀 Features

- **AI Nutritionist**: Get personalized dietary advice and insights based on your goals and preferences.
- **Recipe Discovery**: Browse a vast collection of recipes with advanced filtering capabilities.
- **Smart Shopping List**: Automatically generate shopping lists based on selected recipes.
- **Cooking Assistant**: Step-by-step cooking instructions with interactive guidance.
- **Progress Tracking**: Monitor your nutrition goals and cooking achievements.

## 🛠️ Tech Stack

- **Frontend**: React Native
- **Backend**: Node.js & Express
- **Database**: MongoDB
- **AI/ML**: TensorFlow.js / PyTorch (via API)
- **Authentication**: JWT & OAuth

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- React Native environment
- MongoDB instance

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Misha-Mayskiy/reciper-app.git
   cd reciper-app
   ```

2. **Backend Setup**
   ```bash
   cd backend
   npm install
   # Configure environment variables
   cp .env.example .env
   # Edit .env with your database credentials
   npm start
   ```

3. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   # Start the development server
   npm start
   ```
